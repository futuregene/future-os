# FutureOS

**FutureOS is a desktop AI agent workbench.** It's more than a chat box — you can watch and check the agent's work as it goes: what it read, which commands it ran, which files it changed, what it's waiting for you to approve, and how it should continue.

It's built for people who are actually pushing multi-step tasks forward — software development, research, data analysis, writing, report generation, debugging — all in one place.

> **You stay in control.** Before the agent does anything risky, it stops and asks for your approval. Every piece of work is visible and reviewable.

## What you can do

- **Have an agent conversation** that streams its thinking, calls tools, and shows its work as it happens.
- **Start fast with a Chat**, or bind a folder on your computer as a **Workspace** for real projects.
- **Stay in control** — the agent pauses for your approval before risky actions like writing files or running commands.
- **Review the work** in the right-hand panel: background programs (Runs), file changes (Review), and generated outputs (Artifacts).
- **Use Skills** — installable capability packs the agent uses automatically when they're relevant.
