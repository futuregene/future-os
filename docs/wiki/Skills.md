# Skills

**Skills** are built-in capability packs that give the agent extra, focused abilities — researching the web, reading documents, generating images, and more. The agent uses a skill automatically when it's relevant to what you asked, and you can browse them from **Skill** in the left sidebar.

## Built-in skills

| Skill | What it helps with |
|---|---|
| **Account** | Check your account profile and credit balance, and top up. |
| **Web** | Search the public web and read full pages. |
| **Paper** | Search academic literature (PubMed, ArXiv, DOI) and pull full papers. |
| **Deep research** | Multi-source research with cross-checking, citations, and a written report. |
| **Document** | Turn PDF and Word files into clean, structured text. |
| **Image** | Generate, edit, and analyze images (including reading text from them). |
| **Browser** | Drive a web browser — open pages, click, type, capture screenshots. |
| **Hand-drawn posters** | Create vertical, hand-drawn infographic posters from your content. |
| **Hand-drawn slides** | Create hand-drawn sketchnote slides and assemble them into a PDF. |
| **Subagent** | Run several tasks in parallel for bigger jobs. |
| **Skill creator** | Helps build new custom skills. |

## Using skills

You don't need to turn skills on. Just describe what you want — for example, *"research the latest on X and write me a summary with sources"* or *"turn this PDF into a tidy outline"* — and the agent reaches for the right skill. The **Deep research** workflow also has its own entry under **Research** in the sidebar.
