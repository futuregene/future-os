# 控制台概览

登录 [FutureOS 控制台](https://future-os.cn) 后，左侧为功能菜单，右侧为内容区。

| 菜单 | 功能 |
|---|---|
| 概览 | 账户余额、本月消费、调用次数、消费趋势图 |
| API Key 管理 | 创建和管理 API Key，接入自有系统 |
| 余额充值 | 在线支付、对公转账、兑换码 |
| 团队管理 | 创建团队、邀请成员、分配额度、调整扣费顺序 |
| 费用记录 | 查看模型和工具的逐条消费明细 |

顶部导航栏可随时浏览**技能市场**、**模型列表**和**文档**。

---

# 注册与登录

## 注册

1. 点击右上角「**免费试用**」。
2. 输入邮箱，点击「**发送验证码**」，输入收到的 6 位数字验证码（60 秒有效）。
3. 设置密码：8-20 位，需包含字母和数字。
4. 确认密码，勾选同意协议，点击「**创建账号**」。

## 登录

输入邮箱和密码，点击「**登录**」。勾选「**记住我**」可保持 7 天免登录。

## 忘记密码

点击登录页「**忘记密码？**」→ 输入邮箱 → 获取验证码（10 分钟有效）→ 设置新密码。

# API Key 管理

API Key 用于将 FutureOS 的模型接入你自己的 Agent、脚本或应用。

## 创建

1. 左侧菜单 →「**API Key 管理**」→ 点击「**+ 创建 API Key**」。
2. 输入名称，如"科研助手"或"默认应用"。
3. 点击「**创建**」。

> ⚠️ **密钥仅在创建时展示一次，请立即复制保存。** 关闭弹窗后无法再次查看。

## 使用

在 HTTP 请求 Header 中携带：

```
Authorization: Bearer <你的 API Key>
```

## 管理

- **编辑**：修改名称。
- **删除**：立即失效，不可恢复。

余额永久有效，无过期时间。支持三种充值方式。

## 在线支付

1. 左侧菜单 →「**余额充值**」→ 选择「**在线支付**」。
2. 选择**微信支付**或**支付宝**，输入充值金额。
3. 系统自动计算含服务费的支付总额，点击「**去支付**」完成付款。

# 费用记录

查看模型调用和工具调用的逐条消费明细。

页面顶部展示三项统计卡片：**总消费**、**模型消费**、**工具消费**。

## 模型消费

记录每次模型调用：

| 字段 | 说明 |
|---|---|
| 时间 | 调用时间 |
| 模型 | 使用的模型名称 |
| API Key | 使用的 Key（显示名称，悬停看完整 ID） |
| 入 Token / 出 Token | 输入的 token 数和生成的 token 数 |
| 入 / 出 / 缓存 | 输入费用、输出费用、缓存读取/写入费用 |
| 扣费总额 | 本次调用总费用 |
| 扣费来源 | 主账号 / 团队名称 |
| 状态 | 扣费成功 / 失败 |

## 工具消费

记录每次工具调用：时间、工具名称、API Key、扣费总额、扣费来源、状态。

两表均支持按**日期范围**筛选，点击「**查询**」刷新。

# 团队管理

## 邀请成员

1. 左侧菜单 →「**团队管理**」→ 点击「**+ 邀请成员**」。
2. 输入对方的邮箱地址。
3. 选择预算规则：

| 规则 | 说明 |
|---|---|
| **固定额度** | 一次性分配，有效期结束后余额冻结 |
| **周期重置** | 每天 / 每周 / 每月 / 每季度 / 每年自动重置固定额度 |

4. 设置金额和有效期，点击「**发送邀请**」。

## 管理成员

在「**成员列表**」标签中：

- **调整**：修改成员的额度金额、重置周期或有效期。
- **移除**：将成员从团队中移除。
- **取消**：撤销对方尚未接受的邀请。

## 我加入的团队

在「**我加入的团队**」标签中：

- 处理收到的邀请：**接受**或**拒绝**。
- 已拒绝的邀请可以**删除**记录。
- 点击「**退出团队**」离开已加入的团队。

## 扣费顺序

团队页底部排列个人余额与各团队额度的优先级。点击 **↑ ↓** 按钮调整顺序——扣费时按此顺序依次消耗，前一来源用完后自动切换至下一来源。

# 技能消耗规则

**技能本身不收费。** 但如果技能调用了 FutureOS 提供的工具，则按以下规则计费：

| 工具 | 说明 | 计费规则 |
|---|---|---|
| `web_search` | 调用搜索引擎 | 0.01 元/次 |
| `search_paper` | 搜索学术论文并提取信息 | 0.05 元/次 |
| `get_paper` | 按 PMID/DOI 获取全文 | 0.01 元/次 |
| `fetch_url` | 抓取 URL 内容 | 0.01 元/次 |
| `parse_doc` | PDF/Word 转 Markdown | 0.01 元/页 |
| `browser` | 打开本地网页搜索 | 免费 |
| `image_gen` | 文生图 | 按模型实际计费 |
| `image_edit` | 图片编辑 | 按模型实际计费 |
| `read_image` | 图片识别/OCR | 按模型实际计费 |

工具消费明细可在「**费用记录**」→「**工具消费**」标签中查看。
