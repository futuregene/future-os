# FutureOS

**FutureOS 是一个桌面 AI Agent 工作台。** 它不只是一个聊天框——你可以边看边核对 agent 的工作:它读了什么、跑了哪些命令、改了哪些文件、在等你批准什么、以及接下来该怎么继续。

它面向真正在推进多步骤任务的人——软件开发、研究、数据分析、写作、报告生成、调试——全部在一个地方完成。

> **你始终掌控。** agent 在做有风险的事之前会停下来征求你的批准;所有工作都可查、可核对。

## 你能做什么

- **进行 agent 对话**,它会流式展示思考、调用工具,并实时展示工作过程。
- **用 Chat 快速开始**,或绑定电脑上的一个文件夹作为 **Workspace**,适合真实项目。
- **你始终掌控** —— 在写文件、跑命令等有风险的操作前,agent 会暂停等你批准。
- **核对工作** —— 在右侧面板查看后台任务(Runs)、文件改动(Review)和产出(Artifacts)。
- **使用技能(Skills)** —— 可安装的能力包,相关时 agent 会自动使用。
