# 技能(Skills)

技能是给 agent 的**能力包**。当一个技能已安装、并且它的能力与你正在做的事相关时,agent 会**自动使用**它——你不需要切换模式,也不用手动开启什么。直接描述你的需求即可。

从左栏打开 **Skills** 即可浏览和管理技能。

---

## 浏览与安装

Skills 页有两个标签:

- **Installed(已安装)** —— agent 当前可用的技能。
- **All(全部)** —— 可安装的完整目录。(需要联网。)

用**分类**下拉和**搜索**框查找技能。每个技能会显示名称、描述、版本和分类。点 **Install** 安装,点 **Uninstall** 移除。

---

## 内置技能

目录会不定期更新,你看到的实际列表可能与下表略有不同。常见的内置技能包括:

| 技能 | 用途 |
|---|---|
| **Account** | 你的账户资料、余额与充值。 |
| **Web** | 搜索公网并读取全文。 |
| **Paper** | 检索文献(PubMed / ArXiv / DOI)并获取全文。 |
| **Deep research** | 多源交叉核对,产出带引用的报告。 |
| **Document** | 把 PDF / Word 文件转成结构化文本。 |
| **Image** | 生成、编辑和分析图像——包括读取图中文字。 |
| **Browser** | 驱动网页浏览器:开页、点击、输入、截图。 |
| **Hand-drawn posters** | 生成手绘竖版信息图海报。 |
| **Hand-drawn slides** | 生成手绘草图幻灯并合成 PDF。 |
| **Subagent** | 并行运行多个任务。 |
| **Skill creator** | 帮你制作新技能。 |

> 实际可用的技能、名称和分类都来自在线目录——以应用里 **All(全部)** 标签当前提供的为准。

---

## 如何使用技能

你永远不用手动调用技能。一旦技能已安装:

1. 像往常一样在对话里**描述你的任务**。
2. agent 会在合适时自行选用相关技能。

一如既往,若技能的工作涉及有风险的操作,agent 会先暂停等你批准——见 [[使用 FutureOS|Using-FutureOS]]。
