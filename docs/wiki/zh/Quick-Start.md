# 快速开始

本页带你从全新安装走到第一个回答:登录、开始对话、发消息、选模型、查看工作。

还没装好 FutureOS?先看 [[安装 FutureOS|Installation]]。

---

## 1. 打开并登录

要使用内置模型,先连接你的 FutureGene 账户:

1. 点击窗口左下角的**齿轮图标**(设置)。
2. 进入 **Providers** 页面。
3. 在 **Built-in(内置)** 里找到 **FutureGene**,点 **Connect**。
4. 浏览器会打开进行授权。若没自动打开,应用会给出一个**验证码**和一个**可复制的链接**——用任意浏览器打开链接并输入验证码。
5. 授权成功后,应用会提示已连接。

> 想用自己的模型服务?你也可以改用 OpenAI 兼容或 Anthropic 兼容的 provider。见 [[设置|Settings]]。

---

## 2. 开始对话

有两种方式开始,都在同一个界面里选择:

| | **New Chat** | **Workspace** |
|---|---|---|
| 适用场景 | 快速提问、一次性任务 | 绑定文件夹的真实项目 |
| 需要准备 | 无,直接输入 | 选一个电脑上的文件夹 |

- 点左上角的 **New Chat**,这是最快的开始方式。
- 要做真实项目,选 **Workspace**,再 **Open** 一个电脑上的文件夹。agent 会在这个文件夹里工作。

---

## 3. 发出第一条消息

在底部输入框里输入,按 **Enter** 发送。你会看到:

- 回复**流式**逐字出现。
- **工具活动**:agent 读文件、跑命令等过程会展示出来。
- 遇到有风险的操作(如写文件、跑命令)时会**暂停等你批准**。未经你同意,不会执行任何有风险的操作。

每条消息最多可附 **4 个图片文件** —— 用回形针按钮、粘贴,或把文件拖到输入框上。

---

## 4. 选模型(可选)

**模型选择器**就在输入框里——点它即可为当前会话切换模型。旁边还有**思考级别**控制。

要管理哪些模型出现在列表里,打开**设置 → Models**。见 [[设置|Settings]]。

---

## 5. 查看工作

打开**右侧**面板,核对 agent 正在做什么。它最多有三种视图:

- **Runs** —— agent 正在跑的后台任务,会显示真实命令。你可以停止运行中的任务,或清理已完成的。
- **Review** —— 对 **Workspace**,显示 agent 做出的文件改动:文件列表、统计和 diff。
- **Artifacts** —— 对 **Chat**,收集 agent 生成的产出(报告、摘要、表格、文件)。
