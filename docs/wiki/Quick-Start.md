# Quick Start

From a fresh [[install|Installation]] to your first answer in a few minutes.

## 1. Open FutureOS and sign in

Launch the app. The first time, you'll need an account to use the models.

1. Open **Settings** (the gear at the bottom of the left sidebar) → **Providers**.
2. Under the built-in **FutureGene** provider, click **Connect**.
3. Your browser opens to authorize the sign-in. If it doesn't open automatically, the app shows a verification code and a link you can copy into a browser.
4. Once authorized, you're signed in. You can re-login or log out from the same screen later.

> Prefer your own model provider (OpenAI, Anthropic, …)? You can add one instead — see [[Settings]].

## 2. Start a conversation

FutureOS gives you two ways to work:

- **New Chat** — the quickest way to start. Good for questions and one-off tasks.
- **Workspace** — tie the conversation to a folder on your computer so the agent can work with your real files. Best for projects.

Click **New Chat** in the left sidebar to begin, or open a **Workspace** to point the agent at a folder.

## 3. Send your first message

Type into the box at the bottom and send. You'll see the agent:

- stream its response as it works,
- run tools and show what it's doing,
- and, for anything risky, **pause and ask for your approval** before continuing.

You can attach images to a message (via the attach button, paste, or drag-and-drop) — up to 4 files per turn.

## 4. Pick a model (optional)

The model selector lives right in the message box. You can also manage which models appear, and connect more providers, under **Settings → Models**. See [[Settings]].

## 5. Check the work

The **right-hand panel** lets you inspect what's happening:

- **Runs** — background tasks the agent is running or has finished.
- **Review** — file changes in a Workspace conversation.
- **Artifacts** — files and outputs produced in a Chat.

More on all of this in **[[Using FutureOS|Using-FutureOS]]**.
