# 四个工具演示

## 工具列表

| 工具 | 功能 | 演示结果 |
|------|------|----------|
| `read` | 读取文件 | ✅ 成功读取 hello.txt |
| `bash` | 执行命令 | ✅ 输出了系统信息 |
| `edit` | 编辑文件 | ✅ 追加了新行 |
| `write` | 写入文件 | ✅ 创建了本文件 |

## 总结

这四个工具覆盖了日常开发中的核心操作：
- **read** — 查看文件内容
- **bash** — 运行命令、编译、测试
- **edit** — 精确修改已有文件
- **write** — 创建新文件/覆盖文件
